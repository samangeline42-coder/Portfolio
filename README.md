# Sam Angeline Misa — Personal Portfolio Website

## Description

A clean, responsive personal portfolio website showcasing my profile, skills, projects, education, and contact information. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks required.

## Features

- Responsive layout (mobile, tablet, desktop)
- Smooth scroll navigation
- Animated fade-in sections on scroll
- Clean light aesthetic with consistent styling
- Interactive skill cards and project showcase
- Contact section with clickable email and phone links

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (Intersection Observer API)
- Google Fonts (DM Serif Display, DM Sans)
- Font Awesome 6 (Icons)

## How to Deploy on GitHub Pages

1. Create a new public repository on GitHub (e.g., `portfolio`)
2. Upload `index.html` to the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live in a few minutes!
